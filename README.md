# Game Recommendation Engine

## Live Demo : https://lambent-rabanadas-eb6098.netlify.app/
It is still under development

### TechStack
- Python
- React
- NodeJS
- Material UI

### Installation

**Client**

Within the client folder

```
npm install
npm start
```

**API**

Within the api folder

```
pip install -r requirements.txt
```
then run ```main.py```
